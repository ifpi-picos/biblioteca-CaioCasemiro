# Gerenciamento de biblioteca

**Classes:**
    - Livro:
        1. Título (string)
        2. Autor (string)
        3. Gênero (string)
        4. Disponibilidade (boolean)
    
    - Usuarios:
        1. ID (int)
        2. Nome (string)

    - Emprestimos:
        1. ID (int)
        2. Usuario (Usuario)
        3. Livro (livro) "Livro emprestado"
        4. Data de emprestimo 
        5. Data de devolução
        6. Verificar atraso e calcular multa por atraso
